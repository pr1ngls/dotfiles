# dotfiles
---
This repo contains dotfiles that i primary use nowadays with GNU/Linux via [Arch Linux](https://archlinux.org/)

---
I manage dotfiles using [GNU stow](https://www.gnu.org/software/stow/), so after copying me repo
```
git clone https://github.com/pr1ngls/dotfiles.git
```

You can just type
```
git stow .
```
in dotfiles directory
